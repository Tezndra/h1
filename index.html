Project Stratos: A Resilient LoRa Mesh Communication Network for Disaster Zones

Developed a resilient communication system using a **LoRa mesh network** for disaster zones. This project features a Wi-Fi captive portal for victim connectivity, BLE-based redundancy for enhanced fault tolerance, and an offline geotagged SOS mapping interface.

I integrated TinyML models for intelligent message prioritization and implemented Delay-Tolerant Networking (DTN) protocols to ensure robust connectivity even in the absence of cellular or internet infrastructure.

FARS: Facial Attendance Recognition System

FARS is a real-time facial recognition-based attendance system using a Raspberry Pi 5 and camera. The system is capable of detecting and marking attendance with timestamped entries in daily CSV files. It is designed to automate the attendance process and prevents duplicate entries.

Real-Time Air Quality Monitoring System

This low-cost IoT-based system continuously monitors air quality in real time. It measures PM2.5 and gas concentration (CO, NH, etc.) using environmental sensors. Data is transmitted via Wi-Fi using an ESP8266, and the system provides audible alerts through a buzzer. It also includes AI-based air quality prediction and integrates with Home Assistant for real-time dashboards.
